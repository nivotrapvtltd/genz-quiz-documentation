# Genz Quiz Documentation Site

This folder contains a standalone static documentation website for Genz Quiz.

## Files

- `index.html` - main documentation page
- `styles.css` - documentation styling

## How To Host

You can host this folder on:

- GitHub Pages
- Netlify
- Vercel static hosting
- cPanel / shared hosting
- any static file server

## Simple Upload Method

1. Open the `documentation` folder.
2. Upload `index.html` and `styles.css` to your documentation hosting folder.
3. Open the hosted URL in your browser.

## Recommended Use

Use this documentation site as your buyer help center for:

- installation instructions
- server requirements
- shared hosting setup
- troubleshooting
- payment and AI setup

## Customization

You can edit text directly in `index.html` and update colors or layout in `styles.css`.
